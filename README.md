# Linux System Monitoring Tool (C++)
Developed a menu-driven system tool using C++ for file management, process monitoring, and logging using Linux commands.
This project is a simple Linux-based system utility built using C++.

## Features
- List files in directory (ls)
- Show running processes (ps)
- Create new files
- Log user actions with timestamps

## Technologies Used
- C++
- Linux Commands (ls, ps)
- File Handling

## How to Run

1. Compile:
g++ linux_tool.cpp -o tool

2. Run:
./tool

## Output
Menu-driven system tool for basic file and process operations.
